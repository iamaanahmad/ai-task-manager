AI‑Assisted Task Manager API
📘 Overview
This project is a simple Task Manager REST API built with AI copilots.
It demonstrates how AI can act as a pair programmer, reviewer, tester, and documenter in real development workflows.

🚀 Features
- [x] Create, update, and list tasks

- [x] Track task status (pending/done)

- [x] Add deadlines to tasks

- [x] AI‑generated unit tests for endpoints

- [x] AI‑refactored code for readability and security

🛠️ Tech Stack
Language/Framework: Node.js + Express (or Python + FastAPI — choose one)

Database: SQLite or MongoDB (optional for persistence)

AI Tools: GitHub Copilot, LLM‑assisted debugging and documentation
